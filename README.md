 Fake News Detection System
This project is a machine learning system that detects whether a news article is **fake or real**. It uses NLP techniques and popular ML algorithms such as logistic regression, HAN and SVC to classify news articles.

 Features
 Text preprocessing (cleaning, tokenization)
 Feature extraction using TF-IDF
 Machine Learning models: Logistic Regression, Naive Bayes, XGBoost
 Simple prediction system for news articles

 Tech Stack
 **Python**  
 **Pandas, NumPy**  
 **Scikit-learn, XGBoost, TensorFlow, PyTorch**  
 **Flask / Streamlit** (for future UI)  
 **NLP libraries:** NLTK, Transformers, SentencePiece

Dataset
Contains real and fake news datasets (CSV files)
Data preprocessing handled with Python
